Product 01

LocalAPIClient

A desktop-native HTTP API testing tool built with Tauri and Rust. Designed for developers who value speed, privacy, and a fully offline workflow — without the bloat of cloud-first alternatives.

Tauri v1 Rust JavaScript SQLite
  • Zero sign-up — open and start making requests immediately
  • All data in a single portable .json file on your disk, no cloud sync
  • Full OAuth2 Client Credentials flow with body & header delivery
  • Collections, environment variables, and Postman v2/v2.1 import
  • Bearer, Basic, API Key, and Inherited auth per request
  • Structured NDJSON audit logs written locally every day
  • Sub-1-second cold start, binary under 10 MB
  • 8 built-in themes: Dracula, Nord, Monokai, Solarized & more
